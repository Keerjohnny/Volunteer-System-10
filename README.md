NayePankh Volunteer Registration System

 Project Overview

The NayePankh Volunteer Registration System is a full-stack web application developed to manage volunteer registrations, login, and administration. It provides an easy platform for volunteers to join programs and allows administrators to manage activities.


Features

* Home Page
* Volunteer Registration
* Volunteer Login
* Program Information Page
* Admin Login
* Admin Dashboard
* Success Page after Registration
* Secure Password Hashing using bcrypt
* MySQL Database Integration
* Session Management using express-session



 Technologies Used

 Frontend

* HTML5
* CSS3
* JavaScript

 Backend

* Node.js
* Express.js

 Database

* MySQL

 Packages Used

* express
* mysql2
* dotenv

* express-session
